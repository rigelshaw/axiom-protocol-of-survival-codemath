# AXIOM: Protocol of Survival

AXIOM is a game-like crisis response experience where mathematics is the core mechanic.
Each domain of mathematics represents a different front of a global crisis, and player
decisions directly affect stability, survival, and outcomes.

Built as a solo, frontend-only project for the CODEMATH hackathon.

## Features
- Math-driven gameplay (Algebra, Geometry, Calculus, Number Theory)
- Narrative advisors and real-world crisis framing
- Global stability system affected by decisions
- Cinematic UI with video background and glassmorphism

## Tech Stack
- HTML, Tailwind CSS
- Vanilla JavaScript
- No backend (frontend-only)

## How to Run
Open `index.html` in a browser.
